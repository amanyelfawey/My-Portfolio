Medix

Medix was my graduation project, for which I received an A+. It’s a healthcare web platform that streamlines doctor appointment bookings by specialty and availability and includes an AI-powered tool for brain MRI tumor analysis. Designed with secure authentication and password recovery, Medix is fully responsive for ease across devices. I developed the user interface with React.js, Next.js, and Tailwind CSS, focusing on accessibility and smooth navigation, and created the UI/UX design in Figma.
